# DKS - Remap F to K

Script AutoHotkey v2 qui remappe la touche F vers K,
avec synchronisation de l'appui et du relâchement.

## Prérequis
- AutoHotkey v2 : https://www.autohotkey.com/

## Utilisation
1. Installer AutoHotkey v2
2. Double-cliquer sur `remap.ahk` pour lancer le script
3. Une icône verte apparaît dans la barre des tâches quand le script tourne
